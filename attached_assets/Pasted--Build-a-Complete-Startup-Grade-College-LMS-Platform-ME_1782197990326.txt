# Build a Complete Startup-Grade College LMS Platform (MERN Stack)

IMPORTANT:

Create a professional full-stack MERN application with completely separate frontend and backend folders.

Do NOT create a single-folder project.

Use the following structure:

project-root/
│
├── frontend/
│   ├── src/
│   ├── public/
│   ├── components/
│   ├── pages/
│   ├── layouts/
│   ├── hooks/
│   ├── context/
│   ├── services/
│   ├── routes/
│   ├── assets/
│   ├── App.jsx
│   ├── main.jsx
│   └── package.json
│
├── backend/
│   ├── config/
│   ├── controllers/
│   ├── middleware/
│   ├── models/
│   ├── routes/
│   ├── services/
│   ├── utils/
│   ├── uploads/
│   ├── scripts/
│   ├── server.js
│   └── package.json
│
├── README.md
└── .gitignore

Requirements:

* Frontend and backend must be fully separated.
* Separate package.json for frontend and backend.
* Frontend should run with npm run dev.
* Backend should run with npm run server.
* Use Axios for API communication.
* Use environment variables.
* Configure CORS properly.
* MongoDB logic should only exist in backend.
* Generate complete scalable enterprise-level architecture.

---

## Project Vision

Build a modern, startup-quality College Learning Management System (LMS) with premium UI/UX, advanced dashboards, responsive design, smooth animations, and production-ready architecture.

The UI should look similar to modern SaaS platforms such as:

* Notion
* Linear
* Stripe Dashboard
* Framer
* Vercel

The platform must be fully responsive and optimized for desktop, tablet, and mobile devices.
